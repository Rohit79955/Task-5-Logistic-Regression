# 🔬 Task 5: Breast Cancer Classification using Logistic Regression

📘 **Data Science Internship - EduTech Solutions**

---

### 🎯 Objective
To implement a binary classification machine learning model using Scikit-learn's Logistic Regression to detect and classify medical tumors as malignant or benign.

---

### 📈 Metrics Summary
* **Dataset:** Scikit-learn Breast Cancer Dataset
* **Model Type:** Logistic Regression (Binary Classifier)
* **Deliverable Generated:** `classification_report.txt` containing Precision, Recall, F1-Score, and overall accuracy.

---

### 🛠️ Tools Used
* Python
* Pandas & NumPy
* Scikit-Learn (Datasets, Model Selection, Metrics)
* Matplotlib & Seaborn (Confusion Matrix plotting)
